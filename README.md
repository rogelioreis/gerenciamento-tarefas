### Passo a passo para rodar o programa
#### 3 - No terminal, caminhe até a pasta "src" do programa. Ex:
```shell
cd /caminho/para/projetos/src
```
#### 4 - Execute o codigo:
```shell
python criar_tabelas_tarefas.py
```
#### 5 - Depois de concluído execute o codigo principal:
```shell
python principal.py
```
